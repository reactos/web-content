The ReactOS Project is pleased to announce PWCT as a new participant in the Open Source Support Program.

Programming Without Coding Technology (PWCT) is a set of tools that tries to make GUI programming more accessible, especially for people with little programming experience.

PWCT is focused on visual programming  and comes with a very practical syntax editor. Its program paradigm ranges from imperative to event driven and uses tree hierarchies instead of a drag-and-drop style. The application comes with several samples and the PWCT project has a series of Youtube videos on its main page which helps new users become acclimated with its process.

As anyone who has ever done GUI programming can attest to, making a visual interface is not easy and tools as PWCT helps to make a developer's life much easier.
